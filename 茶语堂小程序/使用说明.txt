茶语堂小程序 - 使用说明
============================

📱 这是一个完整的微信小程序项目

🚀 快速开始:
1. 安装微信开发者工具
2. 打开微信开发者工具
3. 点击"导入项目"
4. 选择本文件夹 (茶语堂小程序)
5. 填入AppID (可选择测试号)
6. 点击"确定"开始开发

📂 项目结构:
├── app.js                 # 小程序入口文件
├── app.json               # 全局配置
├── app.wxss               # 全局样式
├── pages/                 # 页面目录
│   ├── home/             # 京西智谷宣传页
│   ├── order/            # 茶语堂点单页  
│   └── lottery/          # 抽奖活动页
├── utils/                # 工具函数
├── images/               # 图片资源
└── 文档文件...

🎯 主要功能:
✅ 京西智谷北京昇腾公司宣传展示
✅ 茶语堂奶茶点单购物车系统
✅ 抽奖转盘活动
✅ 现代化UI设计
✅ 完整的交互体验

📚 详细文档:
- README.md: 项目介绍
- GETTING_STARTED.md: 快速开始指南  
- PROJECT_OVERVIEW.md: 技术文档
- images/README.md: 图片资源说明

⚠️ 注意事项:
1. 需要添加TabBar图标文件 (详见 images/README.md)
2. 建议在真机上测试完整功能
3. 发布前需配置正式的AppID和服务器域名

🛠️ 技术栈:
- 微信小程序原生框架
- JavaScript ES6+
- WXML + WXSS
