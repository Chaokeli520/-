# 茶语堂小程序 - 项目概览

## 🍵 项目简介

茶语堂小程序是一个集企业宣传、茶饮点单和抽奖活动于一体的微信小程序，专为京西智谷北京昇腾公司打造。通过现代化的UI设计和完整的业务流程，为用户提供便捷的茶饮服务体验。

## 🎯 核心功能

### 1. 京西智谷宣传页 (home)
- **企业展示**: 展示北京昇腾科技有限公司的企业形象
- **轮播图**: 动态展示京西智谷、北京昇腾、科技创新等内容
- **特色介绍**: 展示企业的四大特色（办公环境、技术创新、合作共赢、人才荟萃）
- **茶语堂推广**: 引导用户前往点单页面
- **联系方式**: 提供电话、地址等联系信息

### 2. 茶语堂点单页 (order)
- **分类浏览**: 经典奶茶、水果茶、咖啡系列、特调饮品四大分类
- **商品展示**: 精美的商品卡片，包含图片、名称、描述、价格
- **购物车**: 完整的购物车功能，支持商品数量调整
- **订单提交**: 模拟订单提交流程，引导用户参与抽奖

### 3. 抽奖活动页 (lottery)
- **转盘抽奖**: 精美的旋转抽奖轮盘
- **奖品设置**: 6种奖品，包含概率控制
- **抽奖记录**: 保存用户抽奖历史
- **规则说明**: 详细的抽奖规则和使用说明
- **分享功能**: 支持微信分享增加曝光

## 🎨 设计特色

### 视觉设计
- **主题色彩**: 采用茶饮主题的棕色系色彩搭配
  - 主色: #8B4513 (深棕色)
  - 辅色: #D2691E (橙棕色) 
  - 点缀色: #DAA520 (金色)
- **渐变效果**: 大量使用渐变背景增强视觉效果
- **圆角设计**: 现代化的圆角卡片设计
- **阴影效果**: 适度的阴影提升层次感

### 交互体验
- **流畅动画**: 页面切换和元素出现都有平滑动画
- **反馈机制**: 按钮点击、抽奖等操作都有即时反馈
- **响应式布局**: 适配不同屏幕尺寸
- **手势支持**: 支持滑动、点击等手势操作

## 🏗️ 技术架构

### 开发框架
- **原生微信小程序**: 使用微信小程序原生开发框架
- **语言**: JavaScript ES6+, WXML, WXSS
- **工具**: 微信开发者工具

```

├── app.js                    # 应用主入口
├── app.json                  # 全局配置
```

- **本地存储**: 使用 wx.getStorageSync/wx.setStorageSync
